ItemRepository

class ItemRepository

The ItemRepository class enables you to query and persist item data with the database. The ItemRepository interfaces with DatabaseGateway to save and query raw JSON data. It interfaces with ItemFactory to create items from raw data supplied by DatabaseGateway before returning it to the user in a usable Item object.
